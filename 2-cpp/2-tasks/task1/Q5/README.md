# Q5: Employees Structure

```md
Declare a structure that contains information about employees in a company. These data are:
- Name (struct of first name, middle name, last name)
- Date of Birth (struct of day, month and year)
- Address (struct of street, city, country)
- Contacts (struct of telephone number, mobile number, e-mail address) - Job
- Salary (struct of basic, additional, reductions, taxes)
```
